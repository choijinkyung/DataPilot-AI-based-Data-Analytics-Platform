ReziMate
## **AI 기반 이력서 분석 & 피드백 서비스**

사용자가 PDF 또는 DOCX 형식의 이력서를 업로드하면, AI가 문서를 분석하여 JD에 맞게 이력서를 개선, 포인트를 피드백해주는 웹 애플리케이션입니다.

### **📍프론트엔드**

- React + Vite + Zustand
- SCSS
- React Router v6, Axios

### **📍백엔드**

- Node.js + Express
- PostgreSQL
- JWT (or Keycloak)

### **📍DevOps**

- Docker / Docker Compose
- Kubernetes (K8s)
- Prometheus + Grafana
- GitHub Actions (CI/CD)

### **📍배포**

- AWS EC2 (또는 EKS, ECS)
- S3 + CloudFront (프론트 호스팅)
- AWS RDS
